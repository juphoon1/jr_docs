\doxysubsection{Detailed Description}
M\+TC Profile Interface Functions. 

M\+TC provides profile management for multi-\/user environment. Each user will have his own configuration, contact and log information. They are stored in a sub-\/directory named with user\textquotesingle{}s name.

To load profile of specific user, invoke \mbox{\hyperlink{mtc__cli_8h_ab43e5e1ff28cb39498bae439c546b435}{Mtc\+\_\+\+Cli\+Init}} and \mbox{\hyperlink{mtc__cli_8h_a30ac3a409e026f5a769286a6f38db7bd}{Mtc\+\_\+\+Cli\+Open}}. To load bob\textquotesingle{}s profile in \char`\"{}profiles\char`\"{} directory, use code\+:

\DoxyCodeLine{\mbox{\hyperlink{mtc__cli_8h_ab43e5e1ff28cb39498bae439c546b435}{Mtc\_CliInit}}(\textcolor{stringliteral}{"profiles"});}
\DoxyCodeLine{}
