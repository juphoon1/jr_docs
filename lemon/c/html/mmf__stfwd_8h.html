ZFUNC ZINT Mmf_SSessGetOrigIdPartp ( ZUINT  iSessId,
ZCHAR **  ppcName,
ZCHAR **  ppcUri 
)

Get original identity of participant URI from store and forward session.

The participant is the session initiator who initialized this store and forward session. It will return identity uri if session invitation has P-Asserted-Identity information.

Parameters
[in]iSessIdThe session id.
[in,out]ppcNameThe participant name.
[in,out]ppcUriThe participant URI.

ppcName and ppcUri should be freed by Zos_SysStrFree.

Return values
ZOKGet original participant successfully.
ZFAILEDGet original participant failed.

◆ Mmf_SSessGetPartpType()

ZFUNC ZUINT Mmf_SSessGetPartpType ( ZUINT  iSessId)

MMF im store and forward session get participant type.

Parameters
[in]iSessIdThe session id.
Return values
Typeof im store and forward session, EN_MSF_PARTP_LST_TYPE.

If type is EN_MSF_PARTP_LST_ONE, participant can be got by Mmf_SSessGetPartp. If type is EN_MSF_PARTP_LST_PREDEF_GRP, participant can be got by Mmf_SSessGetPartp. If type is EN_MSF_PARTP_LST_URI_LST, participant can be got by Mmf_SSessGetPartpLstId.
