)

@brlef RLE create push both free text and coordinate file.

Parameters
[in]zCookieThe user cookie.
[in]pcFreeTextThe free text.
[in]dLatitudeThe latitude of user place.
[in]dLongitudeThe longitude of user place.
[in]fRadiusThe radius of user place.
[in]pcLabelThe label string, if ZNULL indicate user's own position, else other position.
[in]pcFileNameThe create file name.

Note: if radius value is equal to 0, the coordinate is a point, if radius value is bigger than 0, the coordinate is a circle.

Return values
ZOKCreate push geolocation information file successfully.
ZFAILEDCreate push geolocation information file failed.
