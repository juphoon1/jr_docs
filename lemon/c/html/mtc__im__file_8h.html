ZCHAR *  pcFileHashVal,

ZVOID *  pData,
ZUINT  iDataSize,
ZUINT  iSrvFlag 
)

MTC Forwarding a file to one participant.

If IM participant receive transfer invitation successfully, GUI will be notified by callback which was set by Mtc_ImCbSetFileAcpted or Mtc_ImCbSetFileRejected.

If IM sending start, GUI will be notified by callback which was set by Mtc_ImCbSetFileSending for report sending file status.

If IM sent successfully, GUI will be notified by callback which was set by Mtc_ImCbSetFileSendOk.

If IM send failed, GUI will be notified by callback which was set by Mtc_ImCbSetFileSendFailed.

While receiving file transfer invitation, GUI will be notified by callback which was set by Mtc_ImCbSetFileRecvIvt.

While IM receiving start, Remote GUI will be notified by callback which was set by Mtc_ImCbSetFileRecving.

While IM received done, Remote GUI will be notified by callback which was set by Mtc_ImCbSetFileRecvDone.

Parameters
[in]zCookieThe user cookie.
[in]pcSubjectThe user subject.
[in]pcUriThe participant URI.
[in]pcFileNameThe file name.
[in]pcFileTypeThe file type, see MTC_IM_FILE_CONT_TXT_PLAIN etc.
[in]iFileSizeThe file size.
[in]pcFileHashValThe file hash value.
[in]iDurationThe file duration.
[in]pDataThe file thumbnail data, thumbnail shall be smaller than 10 kB
[in]iDataSizeThe file thumbnail size, thumbnail shall be smaller than 10 kB
[in]iSrvFlagThe message service flags.
Returns
The file transfer id, forward file transfer invitation successfully, otherwise return ZMAXUINT.

If IM sent successfully, GUI will be notified by callback which was set by Mtc_ImCbSetFileSendOk.

If IM send failed, GUI will be notified by callback which was set by Mtc_ImCbSetFileSendFailed.

While receiving file transfer invitation, GUI will be notified by callback which was set by Mtc_ImCbSetFileRecvIvt.

While IM receiving start, Remote GUI will be notified by callback which was set by Mtc_ImCbSetFileRecving.

While IM received done, Remote GUI will be notified by callback which was set by Mtc_ImCbSetFileRecvDone.

Parameters
[in]zCookieThe user cookie.
[in]pcSubjectThe user subject.
[in]pcUriThe group chat URI.
[in]pcGrpChatIdThe group chat id.
[in]pcFileNameThe file name.
[in]pcFileTypeThe file type, see MTC_IM_FILE_CONT_TXT_PLAIN etc.
[in]iDurationThe file duration.
[in]pDataThe file thumbnail data, thumbnail shall be smaller than 10 kB
[in]iDataSizeThe file thumbnail size, thumbnail shall be smaller than 10 kB
Returns
The file transfer id, send file transfer invitation successfully, otherwise return ZMAXUINT.
