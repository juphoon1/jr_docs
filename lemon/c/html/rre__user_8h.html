Start to Login Client.

If Rre_DbGetUserReg returns ZTRUE, Login result will be notified by GUI pfnShowLoginOk or GUI pfnShowLoginFailed. Otherwise, it means client needn't login to remote server. So it will return immediately and without any GUI callback.

Return values
ZOKClient is logining.
