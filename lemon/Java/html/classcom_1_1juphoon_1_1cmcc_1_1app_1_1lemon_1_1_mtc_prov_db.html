◆ Mtc_ProvDbGetEmgRealm()

static String com.juphoon.cmcc.app.lemon.MtcProvDb.Mtc_ProvDbGetEmgRealm ( )
static

Get emergency server realm.

The general emergency call flow is like below:

  1. No any account has logined
  2. initialize the client
  3. Mtc_CliInit(profile director);
  4. initialize callback functions
  5. mtcCliCbPrint(print callback);
  6. mtcCallCbAlerted(alerted callback);
  7. mtcCallCbTalking(talking callback);
  8. mtcCallCbTermed(terminated callback);
  9. mtcCallCbError(error callback);
  10. Open an null account Mtc_CliOpen(ZNULL)
  11. Start client service Mtc_CliStart()
  12. Get Emergency server info by MtcProvDb.Mtc_ProvDbGetEmgAddr
  13. Set local and server parameters:
    Mtc_CliDbSetSipLclTcpPort(5060);
    Mtc_CliDbSetSipLclTcpPort(5060);
    Mtc_CliDbSetSipRegIp(pcProxyAddr);
    Mtc_CliDbSetSipRegPort(5060);
    Mtc_CliDbSetSipRegRealm(pcProxyRealm);
    Mtc_CliDbSetSipRegTpt(EN_MSP_TPT_UDP);
    Mtc_CliDbSetUserReg(true);
    Mtc_CliDbSetAuthName("unknown");
    Mtc_CliDbSetAuthPass("unknown");
Returns
Default emergency server realm.
