Detailed Description

MTC conference callbacks.

In order to receive MTC conference callbacks, user should implement this interface, then use MtcConfCb.setCallback to register callbacks.

Set callback of indicate conference participant's video stream info updated.

GUI should update conference participant's video stream info. "iConfId" in PFN_MTCCONFCBVIDEOSTRMPTPTUPDT is the id of conference. GUI use it to locate the conference window. "iStrmId" in PFN_MTCCONFCBVIDEOSTRMPTPTUPDT is the video stream id. "pcUri" is the number(not uri) of the participant. If no participant, the URI will be "clear".

Parameters
[in]iConfIdconference Id.
[in]iStrmIdvideo stream id.
[in]urithe number(not uri) of the participant. If no participant, the URI will be "clear". if camera is off, the URI will be "Number:PIC", e.g. "+8616000000313:PIC". if user is removed, the URI wii be "Number1:KICKNumber2", e.g."+8616000000313:KICK1380000000", Number1 is chairman number, number2 is the number of removed participant.

Set callback of indicate conference participant's video stream remove-window info updated.
