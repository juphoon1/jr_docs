int  iMilliSeconds 
)
static

Play ring, and ring type see MtcRingConstants.EN_MTC_RING_TONE_0.

It will play the custom set file if found. Or it will try to play the default ring file. The default ring file path formed by the directory set by MtcRing.Mtc_RingSetDir, the ring name and ".wav" for extension name.

Parameters
[in]iTypeThe ring Type, see MtcRingConstants.EN_MTC_RING_TONE_0.
[in]iMilliSecondsThe playing time length in milliseconds, 0 is for forever.
Return values
MtcCommonConstants::ZOKAudio plays ring successfully.
MtcCommonConstants::ZFAILEDAudio plays ring failed.
See also
MtcRing.Mtc_RingPlayX Mtc_RingStop
