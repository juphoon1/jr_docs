Object  zCookie,
boolean  bOnce 
)
static

Query capability of participant list. Capability query will be operated by SIP OPTION method or SUBSCRIBE method.

While query ok, GUI will be notified by callback which was set by mtcCapCbCapQOk.

While query failed, GUI will be notified by callback which was set by mtcCapCbCapQFailed.

Parameters
[in]iPartpLstIdThe participant list id.
[in]zCookieThe user cookie.
[in]bOnceThe once flag.
Return values
MtcCommonConstants::ZOKcapability query will be running. It will trigger the callback:mtcCapCbCapQOk, mtcCapCbCapQFailed
MtcCommonConstants::ZFAILEDSend message failed.
