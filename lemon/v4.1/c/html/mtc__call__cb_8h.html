◆ Mtc_CallCbSetAlerted()

ZFUNC ZVOID Mtc_CallCbSetAlerted ( PFN_MTCCALLCBALERTED  pfnCb)

Set callback of indicate the callee is ringing.

GUI should update the status of outgoing call window "iSessId" in PFN_MTCCALLCBALERTED is the ID of session. GUI should use it to locate call window.

  • Use Mtc_SessGetEarlyMediaStatus and Mtc_SessHasVideo to check if there is a video early media. If so, use Mtc_SessHasOfferAnswer to check if offer answer exchanged is completed. If so, GUI should invoke Mtc_SessVideoStart to start receive video early media.

  • MTC_CALL_ALERT_RING Alerted by 180 ringing.
  • MTC_CALL_ALERT_QUEUE Alerted by 182 queued.
  • MTC_CALL_ALERT_PROGRESS Alerted by 183 session progress.

◆ Mtc_CallCbSetIncoming()

ZFUNC ZVOID Mtc_CallCbSetIncoming ( PFN_MTCCALLCB  pfnCb)

Set callback of indicate received a new call.

GUI use MTC Session Interfaces, like Mtc_SessGetPeerUri, to get detail information and present those to user. And wait for user action, accept or decline. iSessId in PFN_MTCCALLCB is the ID of session. GUI should use it to get session status.

  • Use Mtc_SessPeerOfferVideo to check if incoming call has video request.
    • Has video, show video call alert window with 3 options
      • accept
      • accept with video
      • decline
    • No Video, show voice call alert window with 2 options
      • accept
      • decline
  • Assign iSessId to alert window. It will be used when invoke Mtc_SessAnswer or Mtc_SessTerm.
  • Get peer information by Mtc_SessGetPeerUri.
Parameters
[in]pfnCbCallback function.

◆ Mtc_CallCbSetMdfyAcpt()

ZFUNC ZVOID Mtc_CallCbSetMdfyAcpt ( PFN_MTCCALLCB  pfnCb)

Set callback of indicate call modifying accepted.
