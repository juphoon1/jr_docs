Detailed Description

Rsd Event Interfaces.

Event provides ability to manage multi Name Value pairs. This buffer is designed for exchange data between modules.

Traditionally, modules exchange data using data structure, which reduce the flexibility of interfaces. When one module's data structure is changed, other modules must also modify its interface. This bring extra workload and bug risk.

In event, data was managed by Name Value pairs. Name is user defined number, like enumeration. Value is the actual value of that Name. It can be number, string or etc. User can also add multi value to the same Name.

Convergency buffer is not thread safe.
