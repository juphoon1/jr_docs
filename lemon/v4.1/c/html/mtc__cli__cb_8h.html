ZFUNC ZVOID Mtc_CliCbSetEvnt ( PFN_MTCCLIEVNT  pfnCb)

Set the service event callback to MTC driver owner. If MTC is running multi-thread mode, MTC and service enablers need be drived by MTC user(mostly is GUI), so MTC will call this callback for convey event to MTC user, MTC user should post event to itself. MTC user then call Mtc_CliDrive for run MTC and service enablers.

Parameters
[in]pfnCbCallback function.
