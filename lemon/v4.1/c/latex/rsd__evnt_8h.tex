\doxysubsection{Detailed Description}
Rsd Event Interfaces. 

Event provides ability to manage multi {\bfseries{Name}} {\bfseries{Value}} pairs. This buffer is designed for exchange data between modules.

Traditionally, modules exchange data using data structure, which reduce the flexibility of interfaces. When one module\textquotesingle{}s data structure is changed, other modules must also modify its interface. This bring extra workload and bug risk.

In event, data was managed by {\bfseries{Name}} {\bfseries{Value}} pairs. {\bfseries{Name}} is user defined number, like enumeration. {\bfseries{Value}} is the actual value of that {\bfseries{Name}}. It can be number, string or etc. User can also add multi value to the same {\bfseries{Name}}.

Convergency buffer is not thread safe.
