Get using default profile management. 

M\+TC provide default profile management. Each account profile is stored in different directory under \char`\"{}profiles\char`\"{} directory. Every Account profile have provision.\+xml etc configuration files.

If client using another profile management, client should close it by \mbox{\hyperlink{mtc__prof__cfg_8h_a3c796ca0e9ed96bd2e28f7b48e3cdffe}{Mtc\+\_\+\+Prof\+Cfg\+Set\+Use\+Dft}}. After \mbox{\hyperlink{mtc__cli_8h_a30ac3a409e026f5a769286a6f38db7bd}{Mtc\+\_\+\+Cli\+Open}}, client should assign database configuration by \mbox{\hyperlink{mtc__cli__db_8h_a53256079d5a07a5dffb3ae2080793034}{Mtc\+\_\+\+Cli\+Db\+Set\+Local\+Ip}} etc interfaces.
