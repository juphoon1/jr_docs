JRSDK Java API Reference for Android 1.0.17
概览
  • JRClient 类用于初始化sdk以及账号登录
  • JRClientCallback 类用于回调登录相关事件
  • JRCall 类用于管理一对一通话
  • JRCallCallback 类用于回调一对一通话相关事件
  • JRMediaDevice 类用于管理设备
  • JRMediaDeviceCallback 类用于管理设备

登录管理

方法 描述
getInstance 获取 JRClient 对象
getCapServerAddress 获取能力开放平台服务器地址
setCapServerAddress 设置能力开放平台服务器地址
startInitWithAppkey 初始化
login 登录
setAutoReLogin 设置是否重试注册
getCurAccount 获取当前open的账号
getCurLoginNumber 获取当前登录的号码
logout 登出
getState 获取当前登录状态
getContext 获取上下文对象
getSDKVersion 获取 sdk 版本号
isNeedUploadCallParameters 是否需要上传通话参数到服务器
setNeedUploadCallParameters 设置是否需要上传通话参数到服务器
getAppKey 获取 AppKey
addCallback 添加回调
removeCallback 删除回调

登录事件

事件 描述
onClientInitResult sdk 初始化结果回调
onClientLoginResult 登录结果回调
onClientLogoutResult 登出回调
onClientStateChange 登录状态变化通知

一对一通话管理

方法 描述
getInstance 获取 JRCall 对象
call 拨打一对一通话
answer 接听一对一电话
mute 开启/取消音频输入静音
hold 开启/取消呼叫保持
switchCamera 切换摄像头
sendDtmf 发送DTMF信息
updateCall 升级或者降级通话
answerUpdate 响应音视频互转的请求
endCall 挂断某路通话
addCallback 添加回调
removeCallback 删除回调
getStatistics 获取统计信息
recordStart 开始音频录制
recordStop 结束音频录制
becomeActive 切换活跃通话

一对一通话事件

事件 描述
onCallItemUpdated 通话状态更新回调
onCallItemRemove 移除通话回调
onCallItemAdd 新增通话回调

设备管理

方法 描述
getInstance 获取 JRMediaDevice 对象
startAudio 启动音频
stopAudio 停止音频
startCamera 开启摄像头
stopCamera 关闭摄像头
switchCamera 切换前后摄像头
startVideo 创建对端画面
stopVideo 关闭对端画面
enableSpeaker 开启关闭扬声器
addCallback 添加回调
removeCallback 删除回调

设备事件

事件 描述
onCameraUpdate 摄像头变化回调
onAudioOutputTypeChange 音频输出变化
onRenderStart 渲染开始
onVideoError 视频渲染错误
