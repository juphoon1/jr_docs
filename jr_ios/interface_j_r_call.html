◆ becomeActive:

- (BOOL) becomeActive: (JRCallItem *)  item

切换活跃通话

调用此方法时，入参 JRCallItem 对象不是活跃的，则会收到 onCallItemUpdate 回调；若入参 JRCallItem 对象已是活跃的，则不会收到通知。

参数
item需要变为活跃状态的通话对象
返回
接口调用结果
  • YES: 接口调用成功
  • NO: 接口调用失败

◆ call:scene:

- (BOOL) call: (NSString *)  peerNumber
scene: (JRCallScene)  scene 

发起呼叫

参数
peerNumber对端号码
scene通话场景，此处只能够选择以下三个场景，若选择了其他场景则直接调用失败
  • JRCallSceneRealTimeWatch : 实时看护场景
  • JRCallSceneVideoCall : 视频通话场景
  • JRCallSceneAudioCall : 音频通话场景
返回
接口调用结果
  • YES: 接口调用成功
  • NO: 接口调用失败
