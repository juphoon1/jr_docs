JRSDK Objective-C API Reference for iOS  1.0.3
概览
  • JRAccount 主要负责账号的增删改查管理
  • JRClient 主要负责SDK初始化及账号的登录登出管理
  • JRClientCallback 用于回调初始化与登录相关事件
  • JRAutoConfig 主要负责通过自动配置获取账户信息
  • JRAutoConfigCallback 用于回调自动配置结果
  • JRMediaDevice 主要负责音视频设备管理
  • JRMediaDeviceCallback 用于回调音视频设备相关事件
  • JRCall 主要负责通话管理
  • JRCallCallback 用于回调通话相关事件

账号管理

方法 描述
createAccount 创建一个账号
setAccount 设置账号配置
getAccountConfig 获取账号配置
deleteAccount 删除账号
getAccountsList 获取所有已存在的账号名称列表

初始化与登录管理

方法 描述
sharedInstance 获取 JRClient 单例对象
startInitSDK 初始化 SDK
login 账号登录
logout 登出当前账号
getLogDirectory 获取日志目录
sdkVersion 获取 SDK 版本

初始化与登录事件

事件 描述
onClientInitResult 初始化SDK结果回调
onLogin 登录结果回调
onLogout 登出回调
onClientStateChange 登录状态变化回调

自动配置管理

方法 描述
sharedInstance 获取 JRAutoConfig 单例对象
startAutoConfig 开始自动配置

自动配置事件

事件 描述
onAutoConfigResult 新增通话回调

音视频设备管理

方法 描述
sharedInstance 获取 JRMediaDevice 单例对象
startCameraVideo 开启摄像头视频渲染
startVideo 开始远端视频渲染
stopVideo 停止视频渲染
rotateVideo 旋转对端画面的角度
startAudio 启用音频
stopAudio 关闭音频
startAudio 开启摄像头
stopAudio 关闭摄像头
switchCamera 切换摄像头
enableSpeaker 开启/关闭扬声器

音视频设备事件

事件 描述
onCameraTypeChange 摄像头切换回调
onAudioOutputTypeChange 音频输出设备切换回调
onRenderStart 渲染开始回调

通话管理

方法 描述
stopVideo 获取 JRCall 单例对象
call 发起呼叫
answer 接听
end 结束通话
mute 静音/解除静音
hold 保持/解除保持
switchCamera 切换摄像头
sendDtmf 发送DTMF
updateCall 发起音视频互转的请求
answerUpdate 响应音频转视频的请求
becomeActive 切换活跃通话
getActiveCallItem 获取当前活跃通话
getStatistics 获得当前通话统计信息

通话事件

事件 描述
onCallItemAdd 增加一路通话回调
onCallItemRemove 移除一路通话回调
onCallItemUpdate 通话状态更新回调
