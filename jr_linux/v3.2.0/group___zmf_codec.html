详细描述

编解码本质是进行连续的数据转换过程。ZMF库提供了注册的方式，将外部的转换过程与内部引擎的业务衔接在一起。

类型定义说明

◆ ZmfCodecOut

typedef void(* ZmfCodecOut) (void *user_data, const char *buf, unsigned length, void *cookie, int bLastFrag)

编解码完成后，通过回调函数导出给使用者

参数
user_datacodecNew 时注册的自定义值
buf编解码输出数据
length编解码输出数据长度
cookiecodecDo 时携带的内部数据
bLastFrag是否为单次编解码操作的最后片段(一次编解码可以分多次进行)
返回
NULL

枚举类型说明

◆ ZmfCodecKey

enum ZmfCodecKey

动态枚举参数，用于 ZmfCodecClass::codecSet, ZmfCodecClass::codecGet

枚举值
ZmfCodecKeyFrame 

value, size 为 int, sizeof(int)，类型是 bool
codecSet 设置当前为关键帧
codecGet 查询，当前为关键帧时返回成功(可选实现)

ZmfCodecBitrate 

value, size 为 int, sizeof(int)，单位是 kpbs
codecSet 设置当前码率

ZmfCodecFramerate 

value, size 为 int, sizeof(int)，单位是 fps
codecSet 设置当前帧率

ZmfAmrBandMode 

value, size 为 int, sizeof(int)
codecSet 设置当前AMR的带宽模式

ZmfCodecWidth 

value, size 为 int, sizeof(int)，单位是 pixel
codecGet 查询当前帧的宽

ZmfCodecHeight 

value, size 为 int, sizeof(int)，单位是 pixel
codecGet 查询当前帧的高

ZmfCodecPacketLoss 

value, size 为 int, sizeof(int)，单位是 %
codecSet 设置当前丢包率(可选实现)

ZmfCodecRtt 

value, size 为 int, sizeof(int)，单位是 ms
codecSet 设置当前双向延时(可选实现)

ZmfCodecErrorConceal 

value, size 为 int, sizeof(int)，类型是 bool
codecGet 查询，支持错误隐藏返回成功

ZmfCodecStream 

value, size 为 int, sizeof(int)，类型是 bool
codecGet 查询，支持流格式返回成功

ZmfCodecH264SPS 

value, size 为 uint8_t[], 字节数，类型是 数据块
codecSet 设置H264的SPS NALU

ZmfCodecH265SPS 

value, size 为 uint8_t[], 字节数，类型是 数据块
codecSet 设置H265的SPS NALU

ZmfCodecH264PPS 

value, size 为 uint8_t[], 字节数，类型是 数据块
codecSet 设置H264的PPS NALU

ZmfCodecH265PPS 

value, size 为 uint8_t[], 字节数，类型是 数据块
codecSet 设置H265的PPS NALU

ZmfCodecH265VPS 

value, size 为 uint8_t[], 字节数，类型是 数据块
