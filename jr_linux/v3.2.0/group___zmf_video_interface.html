◆ Zmf_DesktopCapturePadding()

int Zmf_DesktopCapturePadding ( const char *  pcId,
const int  bPadding 
)

控制屏幕共享是否因8/16对齐而裁剪

图像编码前，会对图像进行裁剪，使分辨率8/16对齐此函数能够打开padding模式，防止屏幕共享因为8/16对齐的裁剪而损失图像信息

参数
pcId采集源ID
bPadding0: 关闭padding功能; 其它: 打开padding功能
返回
int
返回值
0padding模式设置成功
其它padding模式设置失败
前置条件
调用前需要开始屏幕采集 Zmf_VideoCaptureStart
警告
开启padding模式需要同时打开RTP扩展EN_MVD_RTP_EXT_PADDING，否则对端收到的画面会有黑边
