◆ ZmfVideoCaptureCallback

typedef void(* ZmfVideoCaptureCallback) (void *pUser, const char *captureId, int iFace, int iImgAngle, int iCaptureOrient, int *iWidth, int *iHeight, int iPaddingWidth, int iPaddingHeight, unsigned char *buf, ZmfVideoCaptureEncoder *encoder)

回调函数，用于接收采集到的图像数据

参数
pUser通过 Zmf_VideoCaptureAddCallback 注册的用户数据
captureId采集源ID
iFace镜头朝向 ZmfVideoFaceType
iImgAngle图像旋转角度(顺时针)
iCaptureOrient设备固定角度
iWidth输入图像宽，同时作为输出表示裁剪后实际图像宽
iHeight输入图像高，同时作为输出表示裁剪后实际图像高
iPaddingWidth32位补齐后的黑边宽度 Zmf_DesktopCapturePadding
iPaddingHeight32位补齐后的黑边高度 Zmf_DesktopCapturePadding
buf图像I420数据
encoder编码器 ZmfVideoCaptureEncoder
返回
NULL
警告
如果encoder是NULL，输入图像的格式必须是 ZmfPixelFormatI420
参见
ZmfVideoRenderCallback, Zmf_VideoCaptureAddCallback, Zmf_VideoCaptureRemoveCallback

typedef int(* ZmfVideoRenderCallback) (void *pUser, const char *renderId, int sourceType, int iAngle, int iMirror, int *iWidth, int *iHeight, unsigned char *buf, unsigned long timeStamp)

回调函数，用于接收图像渲染数据

参数
pUser通过 Zmf_VideoRenderAddCallback 注册的用户数据
captureId渲染源ID
sourceType视频渲染源类型
参见
ZmfVideoSourceType
参数
iAngle图像角度
iMirror镜像方式 ZmfMirrorType
iWidth图像宽度
iHeight图像高度
buf图像I420数据
返回
int
返回值
>0正在处理渲染数据
0其它
警告
如果buf或iWidth或iHeight等于0，意味着停止渲染，需要同时调用 Zmf_OnVideoRenderRequestRemove

ZmfVideoCaptureCallback
