详细描述

ZMF视频文件格式:
[HEAD][LAYER]*
HEAD -> uint16_t<WIDHT>, uint16_t <HEIGHT>, uint8_t <FPS>
LAYER-> ZmfFileVideoLayer, [Fragment]*
Fragment -> uint32_t<LENGTH>, [DATA]

  • WIDHT, HEIGHT, LENGTH is big endian
  • DATA is byte buffer
