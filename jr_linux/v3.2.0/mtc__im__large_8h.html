ZFUNC ZINT Mtc_ImLMsgGetOrigIdPartp ( ZUINT  iMsgId,
ZCHAR **  ppcName,
ZCHAR **  ppcUri 
)

MTC Get original identity of participant from large message.

The participant is the session initiator who initialized large message session. It will return local URI if local client is the session initiator. It will return identity uri if session invitation has P-Asserted-Identity information.

参数
[in]iMsgIdThe large message id.
[in,out]ppcNameThe participant name.
[in,out]ppcUriThe participant URI.

The caller must copy out parameter, then use.

返回值
ZOKGet original participant successfully.
ZFAILEDGet original participant failed.

◆ Mtc_ImLMsgReply()

ZFUNC ZUINT Mtc_ImLMsgReply ( ZCOOKIE  zCookie,
ZCHAR *  pcUri,
ZCHAR *  pcMsg,
ZUINT  iMsgType,
ZCHAR *  pcConvId,
ZCHAR *  pcContId 
)

MTC Reply to specific standalone large message.

If IM send successfully, GUI will be notified by callback which was set by Mtc_ImCbSetLMsgSendOk.

If IM send failed, GUI will be notified by callback which was set by Mtc_ImCbSetLMsgSendFailed.

While receiving IM(large message), GUI will be notified by callback which was set by Mtc_ImCbSetLMsgRecvMsg.

参数
[in]zCookieThe user cookie.
[in]pcUriThe message receiver SIP URI.
[in]pcMsgThe message body.
[in]iMsgTypeThe message type, see EN_MTC_IM_LMSG_CONT_TYPE.
[in]pcConvIdThe conversation-id.
[in]pcContIdThe contribution-id which reply to.
返回
The large message session id when send large message successfully, otherwise return ZMAXUINT.
