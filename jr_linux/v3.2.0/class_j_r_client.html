◆ driver()

virtual bool JRClient::driver ( void *  zEvntId)
pure virtual

JRClient 驱动服务。

JR SDK使用的是多线程框架，调用接口与回调接口一般为两个线程。 为了线程安全，需要使用该接口将线程转移到主线程。 回调接口参考 JRClientCallback::mtcCliCbEvnt。

参数
[in]zEvntId事件ID.
返回值
true接口调用成功.
false接口调用失败。
