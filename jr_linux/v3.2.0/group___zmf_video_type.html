enum ZmfBeautyEffectType

Video source type.

枚举值

关闭美颜特效

可选的 JSON 参数:
NULL

ZmfCaptureBeauty 

自由选择想要开启的美颜种类以及对应的程度值

可选的 JSON 参数:
"constant": 对比度，范围0-100
"hue": 色相
"saturation": 饱和度
"smooth": 磨皮
"filter": 滤镜
"whiten": 美白
"redden": 红润

可选的 JSON 参数:
NULL

枚举值

虚拟背景

可选的 JSON 参数:
"image": 背景图片路径
"width": 图像宽度
"height": 图像高度

ZmfCaptureErtToRect 

本地采集全景视频

将等角转换为直线
可选的 JSON 参数:
"status": 打开或关闭特效，同时重置所有参数
0: 关闭
1: 打开
"yaw": 几何中心的方位角，范围[0，360]，默认0度
"pitch": 几何中心的高程角度，范围[-180，180]，默认0度
"roll": 围绕几何轴的滚动角度，范围[0，360]，默认0度
"fov": 水平视场角度范围，范围(0，180)，默认90度
警告： fov 太接近0将大大减少图像分辨率
fov 太接近180会大大扭曲图像
建议范围: 45-135
如果没有指定，会依据图像宽度使用2:1的宽高比自动计算
"reset": 重置参数至默认值，有效值：任意数字

ZmfCapturePreferH264 

采集H264类型数据

如果设备支持，将采集H264格式数据，否则使用yuv或mjpg格式
目前，UVC采集(默认CamView)可采用H264，并且摄像头需支持UVC1.5(如Ricoh Theta S)

ZmfCaptureBrightnessNotification 

亮度提示

ZmfDesktopCaptureMouse 

屏幕共享是否采集鼠标

可选的 JSON 参数
"enable": 0: 不采集鼠标； 1(默认): 采集鼠标

ZmfScreenCaptureOptions 

屏幕共享选项

可选的 JSON 参数:
"mouse": true 采集鼠标
"disableEffects": true 屏幕共享时需要关闭桌面特效(如Aero)，目前仅用于Windows
" "disableCAPTUREBLT": true 使用GDI BitBlt()时，从raster-operation移除CAPTUREBLT

ZmfWindowCaptureOptions 

窗口采集共享选项

◆ ZmfRenderMode

enum ZmfRenderMode

视频渲染模式

枚举值
ZmfRenderAuto 

自动选择

当图像的宽高比和渲染区域相匹配，采用 ZmfRenderFullScreen 模式
否则，采用 ZmfRenderFullContent 模式iOS平台会重绘窗口

ZmfRenderMatch 

自动选择

类似 ZmfRenderAuto，但是精度更高，仅支持iOS

ZmfRenderFullScreen 

全屏填充

保持图像宽高比不变的前提下，缩放图像使其铺满渲染窗口(图像可能显示不完全)

ZmfRenderFullContent 

适应内容

保持图像宽高比不变的前提下，缩放图像使其完整显示于渲染窗口(窗口可能存在黑边)

ZmfRenderFullAuto 

自动选择

类似 ZmfRenderAuto，但是不重绘窗口
