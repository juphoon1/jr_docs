◆ Zmf_AudioInitialize()

int Zmf_AudioInitialize ( void *  applicationContext)

初始化ZMF(Zero Media Framework)的音频模块。

ZMF的音频模块负责音频采集、音频播放设备的管理。

参数
applicationContext对于Windows系统, 此参数应该是窗口的句柄，事件通知将被传递给对应窗口。 或者此参数可以是ZmfEventListenCallback类型的回调函数。 对于iOS系统, 此参数应该设置为NULL，它将被忽略。
返回
int
返回值
0ZMF音频模块初始化成功
其它ZMF音频模块初始化失败
前置条件
无
警告
无
参见
Zmf_AudioInitialize2, Zmf_AudioTerminate.

◆ Zmf_AudioInputBoostControl()

int Zmf_AudioInputBoostControl ( int  mode)

设置音频输入设备的麦克风增益控制模式。

仅Windows7系统支持。设置音频输入设备的麦克风增益控制模式。

参数
mode设置的麦克风增益控制模式，参见ZmfBoostMode 。 Off模式：在Zmf_AudioInputStart 接口调用后，系统麦克风增益自动为0， 在Zmf_AudioInputStop 接口调用后，系统麦克风增益恢复。 Manual模式：在Zmf_AudioInputStart 接口调用后,可以通过Zmf_AudioInputGetBoostLevel 接口 和 Zmf_AudioInputSetBoostLevel 接口获取和设置系统麦克风增益。 建议选择off模式。
返回
int
返回值
0调用成功
其它调用失败
前置条件
已成功调用音频初始化接口Zmf_AudioInitialize 。
警告
必须在Zmf_AudioInputStart 之前调用。 在Zmf_AudioInputStop, Zmf_AudioInputStopAll 调用之前，无法改变该模式。
参见
Zmf_AudioInputSetBoostLevel, Zmf_AudioInputGetBoostLevel

int Zmf_AudioInputStart ( const char *  pcId,
int  iSamplingRate,
int  iChannelNumber,
ZmfAecMode  enAecMode,
ZmfAgcMode  enAgcMode 
)

启动指定的音频输入设备。

指定音频输入设备，基于配置信息，启动音频输入设备，开始采集音频数据。

参数
pcIdbuffer指针，指定启动的音频输入设备的Id。
iSamplingRate启动音频输入流的同时，配置的采样率，单位为Hz，0代表自动选择。
iChannelNumber启动音频输入流的同时，配置的通道数，0代表自动选择。
enAecMode启动音频输入流的同时，配置的系统aec模式，参见ZmfAecMode 。
enAgcMode启动音频输入流的同时，配置的系统agc模式，参见ZmfAgcMode 。
返回
int
返回值
0调用成功
其它调用失败
前置条件
已成功调用音频初始化接口Zmf_AudioInitialize 。
已成功获取当前系统可用的音频输入设备总数Zmf_AudioInputGetCount 。
已成功获取指定音频输入设备的设备IdZmf_AudioInputGetName 。
警告
无
参见
