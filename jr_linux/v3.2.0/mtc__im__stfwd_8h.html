◆ Mtc_ImStFwdGetOrigIdPartp()

ZFUNC ZINT Mtc_ImStFwdGetOrigIdPartp ( ZUINT  iSessId,
ZCHAR **  ppcName,
ZCHAR **  ppcUri 
)

Get original identity of participant URI from store and forward session.

The participant is the session initiator who initialized this store and forward session. It will return identity uri if session invitation has P-Asserted-Identity information.

参数
[in]iSessIdThe session id.
[in,out]ppcNameThe participant name.
[in,out]ppcUriThe participant URI.

The caller must copy out parameter, then use.

返回值
ZOKGet original participant successfully.
ZFAILEDGet original participant failed.

◆ Mtc_ImStFwdGetPartpType()

ZFUNC ZUINT Mtc_ImStFwdGetPartpType ( ZUINT  iSessId)

MTC Get participant type from store and forward session.

参数
[in]iSessIdThe session id.
返回
Type of IM store and forward session participant, see EN_MTC_PARTP_LST_TYPE.

If type is EN_MTC_PARTP_LST_ONE, participant can be got by Mtc_ImStFwdGetPartp. If type is EN_MTC_PARTP_LST_PREDEF_GRP, participant can be got by Mtc_ImStFwdGetPartp. If type is EN_MTC_PARTP_LST_URI_LST, participant can be got by Mtc_ImStFwdGetPartpLstId.
