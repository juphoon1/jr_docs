◆ Mtc_RingPlay()

ZFUNC ZINT Mtc_RingPlay ( ZUINT  iType,
ZUINT  iMilliSeconds 
)

Play ring, and ring type see EN_MTC_RING_TYPE.

It will play the custom set file if found. Or it will try to play the default ring file. The default ring file path formed by the directory set by Mtc_RingSetDir, the ring name and ".wav" for extension name.

参数
[in]iTypeThe ring Type, see EN_MTC_RING_TYPE.
[in]iMilliSecondsThe playing time length in milliseconds, 0 is for forever.
返回值
ZOKAudio plays ring successfully.
ZFAILEDAudio plays ring failed.
参见
Mtc_RingPlayX Mtc_RingStop
