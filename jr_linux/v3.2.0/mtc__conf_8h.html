◆ Mtc_ConfCreate()

ZFUNC ZUINT Mtc_ConfCreate ( ZUINT  iType)

MTC conference create.

参数
[in]iTypeThe conference type, EN_MTC_CONF_TYPE.
返回
The id of this new created conference on succeed, otherwise return ZMAXUINT, if return ZMAXUINT, should call Mtc_LcsGetErrCode.

Different with the method of session creation, this function only malloc the resource of conference. If you want to setup or join a conference, you need step ahead by calling other functions.

参见
Mtc_ConfSetup, Mtc_ConfJoin
