MTC notification key for doodle.

宏定义

#define MtcDoodleImageTypeKey   "MtcDoodleImageTypeKey"
 A key whose value is a number object reflecting the type of image, EN_MTC_DOODLE_IMAGE_TYPE.
 
#define MtcDoodleImageNameKey   "MtcDoodleImageNameKey"
 A key whose value is a string object reflecting the image name.
 
#define MtcDoodleImageUriKey   "MtcDoodleImageUriKey"
 A key whose value is a string object reflecting the image URI.
 
#define MtcDoodleResolutionKey   "MtcDoodleResolutionKey"
 A key whose value is a array object contains 2 number object reflecting the resolution in pixels.
 
#define MtcDoodlePositionKey   "MtcDoodlePositionKey"
 A key whose value is a array object contains 2 number object reflecting the position of point, which value is from -1 to 1.
 
#define MtcDoodleActionTypeKey   "MtcDoodleActionTypeKey"
 A key whose value is a number object reflecting the type of action, EN_MTC_DOODLE_ACTION_TYPE.
 
#define MtcDoodleSeqNoKey   "MtcDoodleSeqNoKey"
 A key whose value is a number object reflecting the sequence of action.
 
#define MtcDoodleBrushKey   "MtcDoodleBrushKey"
 A key whose value is a object reflecting the brush's info, contains MtcDoodleWidthKey, MtcDoodleColorKey.
 
#define MtcDoodleWidthKey   "MtcDoodleWidthKey"
 A key whose value is a number object reflecting the width in pixels.
 
#define MtcDoodleColorKey   "MtcDoodleColorKey"
 A key whose value is a number object reflecting the color.
 
#define MtcDoodlePageIdKey   "MtcDoodlePageIdKey"
 A key whose value is a number object reflecting the index of page.
 
#define MtcDoodlePageCountKey   "MtcDoodlePageCountKey"
 A key whose value is a number object reflecting the count of page.
 
#define MtcDoodleUserUriKey   "MtcDoodleUserUriKey"
 A key whose value is a string object reflecting the user's URI.
 
#define MtcDoodleContentKey   "MtcDoodleContentKey"
 A key whose value is a string object reflecting the action's content.
