◆ Mtc_CapQryLst()

ZFUNC ZINT Mtc_CapQryLst ( ZUINT  iPartpLstId,
ZCOOKIE  zCookie,
ZBOOL  bOnce 
)

Query capability of participant list. Capability query will be operated by SIP OPTION method or SUBSCRIBE method.

While query ok, GUI will be notified by callback which was set by Mtc_CapCbSetCapQOk.

While query failed, GUI will be notified by callback which was set by Mtc_CapCbSetCapQFailed.

参数
[in]iPartpLstIdThe participant list id.
[in]zCookieThe user cookie.
[in]bOnceThe once flag.
返回值
ZOKcapability query will be running. It will trigger the callback:Mtc_CapCbSetCapQOk, Mtc_CapCbSetCapQFailed
ZFAILEDSend message failed.
